# analise-redes-mauc-ufc
Análise de conteúdo e métricas de redes sociais do Museu de Arte da UFC (MAUC) utilizando Python (Pandas e Matplotlib). O projeto foca na análise quantitativa da frequência de publicação, mix de formatos e voz da marca (conteúdo institucional vs. cultural/divulgação) durante um período de observação semanal.
